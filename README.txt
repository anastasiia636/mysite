HTML-версии страниц созданы для того, чтобы сайт можно было опубликовать через GitHub Pages и просматривать его со сторонних устройств. GitHub Pages

Основной рабочий вариант проекта остаётся на PHP (файлы `index.php`, `tasks.php`, `coursework.php`), и именно он используется при запуске сайта 
через XAMPP в рамках учебного задания. HTML-файлы не влияют на работу PHP-версии и служат только для удобного внешнего просмотра.
ссылка для просмотра https://anastasiia636.github.io/mysite/



Краткое описание выполненной работы

В ходе выполнения работы был разработан учебный веб-портал на тему веб-разработки.
Сайт имеет физическую и логическую структуру и состоит из нескольких страниц: главной, страницы с заданиями и страницы с описанием курсовой работы.

На сайте реализованы обязательные элементы макета: шапка (header), меню навигации, подвал (footer), контентные блоки и собственный логотип, 
выполненный средствами CSS.
Общие части сайта (header, footer, меню) вынесены в отдельные PHP-файлы и подключаются на всех страницах с помощью `include`, что демонстрирует 
принцип шаблонной структуры и избавляет от дублирования кода.

Главная страница содержит более 300 слов текста и набор изображений, что позволяет оценить работу адаптивной верстки.
Файлы изображений и стили хранятся в отдельных директориях внутри папки `assets`.
Также были созданы статические HTML-версии страниц для публикации проекта на GitHub Pages и возможности просматривать сайт с любых устройств.

Содержимое основных страниц проекта

index.php  главная страница сайта

Содержит:

* общий вводный текст о назначении портала,
* описание принципов адаптивной верстки,
* галерею изображений,
* два информационных блока о структуре сайта и возможностях портала.
  На этой странице размещено более 300 слов и 4 изображения, как требуется по ТЗ.



tasks.php  страница с заданиями

Содержит:

* список примерных учебных заданий по HTML, CSS, адаптивной верстке, PHP и JavaScript,
* каждый пункт оформлен в виде отдельной карточки,
* демонстрирует сеточную верстку и работу контентных блоков.



coursework.php  страница с курсовой работой

Содержит:

* описание темы примерной курсовой,
* цели, задачи и ожидаемые результаты проекта,
* структурированный текст с подзаголовками и списками.
