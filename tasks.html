Задания по курсу

Здесь собраны примеры (!) лабораторных и практических заданий по веб-разработке.

Задание 1. Простая HTML-страница

Создайте страницу с заголовком, несколькими абзацами текста и изображением. Проверьте корректность отображения в разных браузерах.

Задание 2. Адаптивная верстка

Сверстайте блок из текста и картинок так, чтобы на широком экране элементы располагались в две колонки, а на телефоне — в одну.

Задание 3. Общие части сайта на PHP

Вынесите шапку, меню и подвал в отдельные файлы PHP и подключайте их на всех страницах портала.

Задание 4. Работа с JavaScript

Добавьте на сайт небольшой скрипт: например, обработку клика по кнопке, плавную прокрутку или вывод текущей даты.
