Задания по курсу

Здесь собраны примеры(!) лабораторных и практических заданий по веб-разработке.

Задание 1. Верстка сайта

Нужно создать адаптивный сайт на любую тему с единой стилистикой и продуманной структурой. В нём должны быть шапка, подвал, меню, контент и логотип (30–100 px, сделанный с помощью CSS). Главная страница должна содержать не менее 300 слов текста и 4–5 изображений. Если сайт одностраничный, должны быть переходы на страницы с заданиями и курсовой, где также сохраняются основные элементы макета. Верстка — любым способом, кроме табличной. Макет можно сначала сделать в Figma или Axure. Готовый сайт нужно выложить на GitHub Pages.
